¿Qué es HTML?

HTML es el lenguaje utilizado para estructurar el contenido de una página web. Permite definir encabezados, párrafos, imágenes y otros elementos fundamentales.

¿Por qué es importante el HTML5?

HTML5 incorporó nuevas etiquetas semánticas que mejoran la accesibilidad, el SEO y la organización del contenido.
